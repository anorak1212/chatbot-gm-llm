# Asistente GM — Best Practices de Interior Trim Design

Soy tu asistente especializado en las **7 Best Practices (BP1-BP7)** de diseno de trim interior de puertas para General Motors.

## Que puedo hacer

- Responder preguntas sobre requisitos dimensionales, mecanicos, de seguridad y logistica
- Listar todas las best practices disponibles
- Consultar detalles especificos de cada BP (torques, clearances, dimensiones)
- Resolver dudas sobre interfaces entre componentes

- **"Hola"** — Para iniciar una conversacion

- **"Cual es el torque del tirador?"** — Para preguntas especificas
